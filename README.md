# 🐤 Flappy Bird - Pygame Edition

A simple Flappy Bird clone made with Python and Pygame.

## 🕹️ How to Play
- Press **Spacebar** to flap.
- Avoid hitting pipes and ground.
- Survive as long as possible!

## 🔧 Requirements
- Python 3.x
- Pygame

## 🚀 Run the Game

```bash
pip install pygame
python flappy_bird.py
